# Short_BCH_quasi-BP_decoding

## Project Overview

This project implements quasi-BP decoding algorithms for short BCH codes. The codebase is organized into three main modules:

| Module | Purpose | Status |
|--------|---------|--------|
| `BCH_127_Testing` | **Main module** for decoding and evaluation | 🟢 Primary focus |
| `BCH_127_Training` | Training-related code | 🔴 **Almost unused** in current project |
| `Testing_data_gen_127` | Generates and saves raw received sequences for decoding | 🟡 Supporting module |

## ⚠️ Strong Recommendations

Before working with any module, **please read its corresponding `Readme.md` file** for:
- Module-specific setup instructions
- Configuration details
- Usage examples and dependencies
